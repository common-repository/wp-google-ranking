== Description ==

<strong>WP Google Ranking</strong> is a daily cron service plugin for Wordpress, that helps you to track and check the <strong>Google SERPS's fluctuations</strong>.<br><br>
It checks every day your <strong>SERP position</strong> in several Google geo-servers:
<ul><li>google.com</li><li>google.co.uk</li><li>google.de</li><li>google.es</li><li>google.it</li><li>google.fr</li><li>and more coming soon :)</li></ul><br>
<a href="http://twitter.com/gufastian" style="text-decoration:underline;color:red">Follow me on twitter!</a><br>
If you are in trouble with the plugin, <a title="Contatti" href="http://www.sebastianomontino.com/contatti/" target="_self">contact me</a>!

== Installation ==

<br>Just go in the <strong>"Options" menu</strong> under "WP Google Ranking" and set your: <br><ul><li><strong>Domain</strong> (ex. <i>androidiani.com</i>)</li><li><strong>Keyword</strong> to check on Google</li><li><strong>Number of Pages to check</strong> (from 1 to n)</li><li>The local <strong>Search Engine</strong> :)</li></ul>
